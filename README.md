  <h2>Hunspell Dictionary API</h2>
  <p>This is a simple API that uses <a href="https://github.com/LibreOffice/dictionaries" target="_blank">  LibreOffice dictionaries </a> to verify if a given word exists in a specific language and variation.</p>
  <p>To check supported languages, go to <a href="https://dicio.mrchuw.com.br/languages">/languages</a>.</p>
